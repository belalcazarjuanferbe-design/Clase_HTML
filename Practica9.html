Lista de definiciones

Python
Lenguaje de programación interpretado, de alto nivel y generalista.

JavaScript
Lenguaje de programación interpretado, principalmente utilizado en el lado del cliente para crear páginas web dinámicas.

C#
Lenguaje de programación desarrollado por Microsoft, utilizado para crear aplicaciones .NET.

Java
Lenguaje de programación orientado a objetos, ampliamente utilizado en el desarrollo de aplicaciones empresariales.

C++
Lenguaje de programación compilado, derivado de C, utilizado para el desarrollo de software de alto rendimiento.

PHP
Lenguaje de programación server-side, ampliamente utilizado para el desarrollo web.
